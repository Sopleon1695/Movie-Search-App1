About Geek Movie

Geek Movie is a movie search web application that allows users to search for movies and view movie information such as posters, titles, and release years.


This website was created as a web development project using HTML, CSS, JavaScript, and the OMDb API.


The goal of this project is to demonstrate how APIs can be integrated into web applications to display real-time movie data.
